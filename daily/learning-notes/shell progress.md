
### **Shell scripting basics**
    
    - Looping over a **string** using `for`
        
    - Understanding how the shell expands and executes commands
        
- **File & content viewing**
    
    - `cat` – dump file content to stdout
        
    - `more` – paginated viewing (older, simple pager)
        
    - `vi` – basic file editing and navigation
        
- **Filesystem & user context**
    
    - `ls` – list files/directories
        
    - `pwd` – identify current working directory
        
    - `id` – check user ID, group ID, and privileges
        
- **Networking**
    
    - `nc` (netcat) – connect to ports, test listening services
        
    - Understanding port connectivity features
