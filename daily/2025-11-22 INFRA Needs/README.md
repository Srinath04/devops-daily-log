Learning Reflection

*_Saying_* 
A good Infrastructure should be cattle, not pets.

*_Meaning_* 
-  **Stop treating servers like pets that we nurse back to health.  Start treating them like uniform cattle we can replace automatically.**

Whats better if we follow the above two ?
*_Gaining_*
- No config drift 
- Auto-healing is easy
- Scaling is trivial
- Zero downtime deployments
	- BG, Rolling updates, canary 
- Works perfectly with cloud
